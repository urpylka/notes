### Работа с маршрутами

Для просмотра маршрутов на macOS используется утилита `netstat`

* `-r` to show routes
* `-n` to not resolve IPs to hostnames

```bash
netstat -rn

Для добавления/удаления машрута:

sudo route -n add x.x.x.x/24 x.x.x.x
sudo route -n add 192.168.10.0/24 10.0.247.43

sudo route -n delete x.x.x.x/24 x.x.x.x

Изменение раздела с помощью diskutil

diskutil list

# изменение размера раздела
diskutil resizeVolume
diskutil resizeVolume disk0s2 350g
diskutil resizeVolume disk_identifier partition_size second_partition_format second_partition_name second_partition_size

Аналог losetup в macOS

# подключение образа
hdiutil attach build_server_backup.img

# создание образа notes.dmg
hdiutil create -srcfolder ~/Documents/Notes notes

# создание зашифрованного, расширяемого образа
hdiutil create ~/Documents/TopSecret -type SPARSEBUNDLE \
  -size 100m -fs HFS+J -volname TopSecret -encryption
