Ускорение загрузки: systemd-analyze

Это краткая заметка по анализу времени загрузки служб ОС.

В Linux дистрибутивах в которых используется systemd-analyze есть возможность проанализировать сколько какие службы загружались по времени:

systemd-analyze critical-chain

# если включен графический интерфейс, то его можно выключить
sudo systemctl disable graphical.target
